Ubuntu VBox Proxychains

Automatización de despliegue para administradores de sistemas (ASIR)

🚀 El Proyecto

Este repositorio proporciona un flujo de trabajo automatizado para instalar y configurar VirtualBox junto con ProxyChains en entornos Ubuntu/Debian.

Ideal para despliegues rápidos donde la seguridad y el anonimato del tráfico de red son prioridades técnicas.

curl -sSL https://raw.githubusercontent.com/Haplee/ubuntu-vbox-proxychains-setup/main/install.sh | bash

🛠️ Características Core

Modularidad

Scripts divididos por funciones: comprobaciones pre-vuelo, gestión de dependencias y configuración de kernel.

Hardening de Red

Integración nativa con ProxyChains4 para tunnelring dinámico de aplicaciones.

Robustez

Manejo avanzado de errores, logs detallados y estados idempotentes en cada ejecución.

Fácil Limpieza

Script de desinstalación completo que revierte todos los cambios realizados en el sistema.

📂 Arquitectura

La solución se organiza en una estructura limpia y fácil de mantener:

  • modules/ - Lógica de instalación separada por capas.
  • config/ - Plantillas para ProxyChains y alias de shell.
  • tests/ - Suite de verificación de entorno.
  • docs/ - Esta documentación estática.

📖 Uso Rápido

Clona el repositorio y lanza el instalador principal:

git clone https://github.com/Haplee/ubuntu-vbox-proxychains-setup.git
cd ubuntu-vbox-proxychains-setup
chmod +x install.sh
./install.sh
