Ubuntu · VirtualBox · Proxychains

Automatiza tu entorno
de virtualización
con un solo comando.

Script modular para instalar y configurar VirtualBox + Proxychains en Ubuntu. Detección automática del sistema, soporte Secure Boot y configuración de proxy lista para usar.

Instalación rápida Ver módulos
bash — install.sh
$ git clone https://github.com/user/vbox-setup.git
$ cd vbox-setup
$ chmod +x install.sh
$ sudo ./install.sh
✓ Sistema detectado: Ubuntu 22.04 LTS (x86_64)
✓ Instalando dependencias...

Instalación

Tres formas de ejecutar el setup según tus necesidades.

⚡

Instalación completa

Ejecuta todos los módulos en orden: preflight, detección, dependencias, VirtualBox, kernel modules, Secure Boot, extension pack y proxychains.

sudo ./install.sh
🧩

Módulo individual

Ejecuta solo el módulo que necesitas. Útil para reinstalaciones parciales o para depurar una etapa específica del proceso.

bash modules/archivo.sh
🧹

Desinstalación

Elimina todos los componentes instalados: VirtualBox, extension pack, proxychains y las configuraciones asociadas del sistema.

sudo ./uninstall.sh
📋

Requisitos previos

✓
Ubuntu 20.04+

Compatible con 20.04 LTS, 22.04 LTS y 24.04 LTS en arquitectura x86_64.

✓
Permisos sudo

Se requieren privilegios de administrador para la instalación de paquetes y módulos del kernel.

✓
Conexión a internet

Necesaria para descargar VirtualBox, el Extension Pack y las dependencias apt.

✓
4 GB RAM mínimo

Recomendado 8 GB para ejecutar máquinas virtuales de forma cómoda.

Módulos del sistema

11 módulos ejecutados secuencialmente. Cada uno es independiente y puede relanzarse por separado.

00

preflight.sh

Comprobaciones previas: verifica que el sistema cumple los requisitos mínimos antes de iniciar la instalación.

preflight
01

detect_system.sh

Detecta la distribución, versión del kernel, arquitectura y si el sistema arranca con UEFI/Secure Boot.

detección
02

install_deps.sh

Instala todas las dependencias necesarias vía apt: build-essential, dkms, linux-headers y otras librerías.

dependencias
03

virtualbox.sh

Agrega el repositorio oficial de Oracle, importa la clave GPG e instala la versión estable de VirtualBox.

core
04

kernel_modules.sh

Compila e instala los módulos del kernel vboxdrv, vboxnetflt y vboxnetadp usando DKMS.

core
05

secure_boot.sh

Genera y enrolla un MOK (Machine Owner Key) para firmar los módulos del kernel cuando Secure Boot está activo.

seguridad
06

extension_pack.sh

Descarga e instala el VirtualBox Extension Pack compatible con la versión instalada (USB 3.0, RDP, cifrado).

extras
07

proxychains.sh

Instala proxychains-ng y configura el archivo base para enrutar conexiones a través del proxy definido.

proxy
08

proxy_config.sh

Aplica la configuración de proxy del archivo config/defaults.conf al sistema y a proxychains.

proxy
09

aliases.sh

Agrega aliases útiles al .bashrc o .zshrc: atajos para vboxmanage, proxychains y comandos frecuentes.

shell
10

verify.sh

Verifica que la instalación se completó correctamente: comprueba servicios, módulos cargados y conectividad.

verificación
