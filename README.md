---

## 🚀 Instalación Ultra-Rápida

Ejecuta el siguiente comando para iniciar el asistente de instalación automática:

```bash
curl -fsSL https://raw.githubusercontent.com/Haplee/ubuntu-vbox-proxychains-setup/main/install.sh | sudo bash
```

---

## ✨ Características Principales

- **📦 VirtualBox Core**: Instalación desde repositorios oficiales de Oracle.
- **🔐 Secure Boot Ready**: Firma automática de módulos de kernel (MOK) para compatibilidad total con Secure Boot.
- **🧩 Extension Pack**: Instalación automática de las extensiones de VirtualBox.
- **🌐 Network Hardening**: Configuración de ProxyChains4 preconfigurado para túneles dinámicos.
- **⚡ Shell Optimization**: Aliases automáticos para `bash`, `zsh` y `fish`.
- **🛠️ Modularidad**: Código limpio, modular e idempotente.

## 🛠️ Uso Local

Si prefieres clonar el repositorio y revisar el código antes de ejecutarlo:

```bash

cd ubuntu-vbox-proxychains-setup

# Dar permisos y ejecutar
chmod +x install.sh
./install.sh
```
