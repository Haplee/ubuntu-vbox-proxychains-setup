# Ubuntu VirtualBox & ProxyChains Setup

Script de automatización total para instalar y configurar VirtualBox y ProxyChains en Ubuntu.

[📺 Ver documentación web (GitHub Pages)](https://haplee.github.io/ubuntu-vbox-proxychains-setup/)


## Compatibilidad
- Ubuntu 18.04 LTS (Bionic)
- Ubuntu 20.04 LTS (Focal)
- Ubuntu 22.04 LTS (Jammy)
- Ubuntu 24.04 LTS (Noble)

## Instalación Rápida
```bash
curl -fsSL https://raw.githubusercontent.com/Haplee/ubuntu-vbox-proxychains-setup/main/install.sh | sudo bash
```

## Características
- [x] Instalación automática de VirtualBox desde repositorios oficiales de Oracle.
- [x] Firma de módulos automática para Secure Boot (MOK).
- [x] Instalación de Extension Pack.
- [x] Configuración de ProxyChains con soporte para Tor.
- [x] Aliases automáticos para Bash, Zsh y Fish.
